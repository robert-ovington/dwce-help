Deliveries

Record aggregate and waste deliveries directly in the app. All delivery records are stored digitally and can be reviewed, edited, and exported as a PDF.

Submitting an Aggregate Delivery

Use this screen to record when aggregate materials are delivered to a job. You can record multiple deliveries for the same job on the same day — for example, if a lorry makes several runs.

  1. From the main menu, tap Deliveries → Aggregate Delivery.
  2. Select the Job / Project the delivery is for from the dropdown list.
  3. The date will default to today. Change it if recording a delivery from a different date.
  4. Select the material type (e.g. Type 1 sub-base, sharp sand, gravel).
  5. Enter the quantity and unit (e.g. tonnes, m³).
  6. Enter the supplier name or select from the list.
  7. Enter the delivery ticket / docket number if available.
  8. Add any relevant notes in the comments field.
  9. Tap Save Delivery.

Adding multiple deliveries for the same job

After saving the first delivery, you can immediately add another for the same job and date:

  1. After saving, tap + Add Another Delivery.
  2. The job and date will pre-fill from your previous entry.
  3. Enter the details for the next delivery and tap Save Delivery.
  4. Repeat for as many deliveries as required.
✅

Record each delivery as it arrives on site, rather than at the end of the day. This ensures accurate quantities and delivery times.

Submitting a Waste Delivery

A waste delivery record is created each time a waste load leaves the site. Each record covers a single waste type. The app uses your saved signature and also allows for capturing the signature of the waste facility receiving the load.

  1. From the main menu, tap Deliveries → Waste Delivery.
  2. Select the Job / Project the waste is coming from.
  3. The date will default to today.
  4. Select the waste type from the list (one type per record — e.g. inert soil, mixed construction waste, concrete).
  5. Enter the quantity and unit.
  6. Enter the waste carrier name and licence number if required.
  7. Enter the name of the waste facility where the load is being taken.
  8. Enter the consignment / transfer note number if applicable.

Driver signature

The app uses your saved signature for waste delivery records. Your signature is set up once when your account is configured — it will be applied automatically.

  • If your saved signature is displayed and correct, it will be included automatically
  • If your signature needs updating, contact the office to have it reset

Waste facility signature

If the waste facility requires a signature from their staff to confirm receipt, you can capture it in the app:

  1. At the waste facility, tap Add Facility Signature.
  2. Hand your phone to the facility operative.
  3. They sign directly on the screen using their finger.
  4. Tap Accept Signature to save it.
  1. Review all details and tap Save Delivery.
⚠️

Each waste delivery record must have a single waste type. If you are carrying mixed waste types in separate loads, create a separate record for each.

My Deliveries — Viewing, Editing, Deleting & PDF

My Deliveries is a complete history of all delivery records you have submitted. You can filter the list, edit or delete records, and export any delivery as a PDF.

  1. From the main menu, tap Deliveries → My Deliveries.
  2. Your delivery history is displayed, most recent at the top.

Filtering and searching

Use the filter options at the top of the screen to narrow down the list:

  • Date range — select a start and end date to view deliveries within a period
  • Delivery type — filter by Aggregate or Waste
  • Job / Project — view deliveries for a specific job

Viewing a delivery record

Tap any entry in the list to open the full details of that delivery record.

Editing a delivery

  1. Open the delivery record you want to change.
  2. Tap the Edit button.
  3. Update the relevant fields.
  4. Tap Save to confirm.
ℹ️

Delivery records can only be edited within a set time window. If the Edit button is not available, contact the office for an amendment.

Deleting a delivery

  1. Open the delivery record you want to delete.
  2. Tap the Delete button (bin icon).
  3. Confirm the deletion in the prompt that appears.
⚠️

Deletion is permanent. Only delete a record if it was entered in error.

Exporting to PDF

Any delivery record can be exported as a PDF — useful for providing a paper copy to a client or waste facility.

  1. Open the delivery record.
  2. Tap Export PDF or the share icon.
  3. The PDF will be generated and you can save it, print it, or share it via email or messaging apps.
✅

PDF exports include all delivery details, both signatures (where captured), and a timestamp. They are suitable for use as waste transfer notes or delivery dockets.
